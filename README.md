# TaskFlow 🚀

A modern full-stack task management web application built using the MERN Stack with beautiful animations, interactive UI, and real-time task management features.

---

# 🌟 Features

✅ Add Tasks  
✅ Edit Tasks  
✅ Delete Tasks  
✅ Mark Tasks as Completed  
✅ Task Filters (All / Pending / Completed)  
✅ Enter Key Support  
✅ Toast Notifications  
✅ Confetti Celebration 🎉  
✅ Smooth Framer Motion Animations  
✅ Glassmorphism UI  
✅ Floating Animated Background  
✅ Responsive Design  
✅ REST API Integration  
✅ MongoDB Database Integration  

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Framer Motion
- Axios
- React Hot Toast
- React Confetti

## Backend
- Node.js
- Express.js

## Database
- MongoDB Atlas
- Mongoose
